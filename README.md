# smart-watch
Учебный проект. Сайт по подбору пульсометров.

При создании проекта были применены:

- Flexbox
- Формы с маской ввода, валидацией и отправкой на почту с помощью phpmailer
- Начальные навыки JS
- CSS слайдер
- Интерактивные карты
- CSS - анимации
